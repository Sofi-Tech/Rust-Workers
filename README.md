# Rust-Workers

Blazingly Fast..!

## Contribution Guideline

Create an `.env` file in the root of the project and add your mongodb uri to it.

```env
MONGODB_URI=<your-mongo-uri>
```

### Running

```bash
cargo run
```

### Testing

```bash
cargo test
```

### Benchmarking

```bash
cargo bench
```

### Formatting and Linting

```bash
cargo fmt
cargo clippy
```
